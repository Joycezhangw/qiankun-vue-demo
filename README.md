# hmall-micro

#### 介绍

微前端,qiankun

该项目所有，都使用 `vue-cli 3` 下创建的`vue`项目，单独运行项目命令，请参照 [Vue CLI 官方文档](https://cli.vuejs.org/zh/guide/prototyping.html)

#### 软件架构

`vue` `qiankun`


#### 安装微应用依赖教程

 `npm run hm-install-all`

#### 项目启动开发环境

`npm run hm-serve-all`

#### 打包所有项目

`npm run hm-build-all`
